# 审判日，以及 AI 行业集体进入成年期

*2026.04.21 — 04.27 · 第 18 周*

4 月 27 号，加州奥克兰联邦法院里，9 个普通人坐在陪审席上。

他们要裁决的事情，用一句话概括就是，一个叫 Elon Musk 的人说另一个叫 Sam Altman 的人偷走了一个慈善机构，然后把它变成了一家估值 8520 亿美元的公司。

索赔金额 1340 亿美元。Musk 说一分钱不要自己留。

这个画面很有意思。两个曾经坐在同一张桌上讨论「如何拯救人类」的人，现在坐在法庭的两侧，隔着律师团互相指着对方说「你是骗子」。Musk 在 X 上发帖，原话是，「骗子 Altman 和 Stockman 偷走了一个慈善机构。句号。」

坦率的讲，这场审判的法律结果可能并不重要。

26 项诉求砍到只剩 2 项，欺诈指控自己撤了，剩下的「不当得利」和「违反慈善信托」要赢都挺难。但审判过程中会曝光的东西，那些内部邮件、董事会记录、股权变更文件，才是真正的炸弹。

一个号称为「全人类福祉」成立的非营利组织，8 年之内变成了 2 万亿估值的商业帝国。这个故事怎么发生的，每一步的决策是什么，谁签了字，谁反对过，谁沉默了。

这些东西一旦公开，影响的不只是 OpenAI。

整个 AI 行业的公司治理、非营利转型的法律边界、创始人承诺的约束力，全都会被重新定义。

我跟你说，这比任何产品发布都值得关注。

---

但法庭之外发生的事情，密度甚至比法庭之内还高。

同一周，Google 宣布最高 400 亿美元投资 Anthropic。

400 亿。

这不是风投。风投讲回报率、讲退出机制。这 400 亿的结构是这样的，先打 100 亿现金加算力，Anthropic 达成特定目标后追加到 400 亿。Bloomberg 报道说这笔钱的本质是「供应商融资」，Google 用注资换取 Anthropic 对其 TPU 算力基础设施的长期依赖。

你想想这个逻辑。Anthropic 年化收入刚突破 300 亿美元超过 OpenAI，Claude 生态从 Code 到 Design 全面铺开，势头猛得不行。Google 不跟它竞争，而是直接投它。

为什么？

因为 Google 想的不是「打败 Anthropic」，而是「无论 Anthropic 赢还是 OpenAI 赢，我的 TPU 都是底层基础设施」。

这才是巨头级别的思维。不下场打架，把擂台建好，收门票。

同一周另一边，SpaceX（已经跟 xAI 合并了）拿到了 600 亿美元收购 Cursor 的期权。不是直接收购，是期权，年底前可以选择买还是不买。如果不买，付 100 亿合作费。

600 亿买一个代码编辑器？

但你仔细想想就知道这个钱买的不是编辑器。Cursor 有 300 万日活开发者，DAU/MAU 72%，这个留存率在 SaaS 历史上几乎没有先例。Musk 在内部备忘录里说得很清楚，这是「打造地球上最具雄心的垂直整合创新引擎」的一部分。AI 编程工具、xAI 大模型、SpaceX 算力中心、星链通信、X 平台，全串起来。

Cursor 拿到这笔交易之后，第一个动作是什么？摆脱对 Anthropic 和 OpenAI 模型的依赖。用 xAI 的超级计算机训练自有模型。

你看到了吗。

Google 400 亿投 Anthropic，锁定算力订单。SpaceX 600 亿锁 Cursor，锁定开发者入口。Musk 和 Altman 在法庭上撕，但真正的战争不在法庭。

AI 的竞争已经不是「谁的模型更强」了。

是「谁拥有完整的基础设施闭环」。模型、算力、分发渠道、开发者生态、企业客户。缺任何一环，你就不是参赛选手，你只是供应商。

---

说完巨头的排列组合，聊聊这周最让我觉得兴奋和不安的两件事。

先说兴奋的。

4 月 23 号，OpenAI 发了 GPT-5.5。代号「Spud」，就是土豆的意思。距离 5.4 发布不到两个月。

核心数据，Terminal-Bench 编程 82.7% 登顶 SOTA，超过 Claude Opus 4.7 的 69.4%。100 万 token 上下文窗口。ARC-AGI-2 抽象推理 85.0%，去年这个测试的最高分还不到 50%。

100 万 token 是什么概念。一整部哈利波特七本放进去还有余量。你可以把整个项目代码库、所有设计文档、历史邮件记录一股脑全塞进去，然后让它理解上下文帮你干活。

API 定价 5 美元输入、30 美元输出每百万 token。一个游戏推广视频的脚本，AI 辅助成本不到 1 美分。

这是什么速度。

但真正让我在意的不是这些数字。是 Altman 说的一句话，「更少引导下完成更多」。

怎么理解呢。之前的模型你得手把手教它，prompt 要写得很精确，稍微模糊一点它就跑偏。5.5 的变化是，你说个大概意思，它就能猜到你想要什么。

从「工具」到「助手」的质变，可能就在这一步。

但这周还发生了一件事，让我从兴奋直接跌到不安。

4 月 26 号，PocketOS 的创始人 Jer Crane 公开披露了一个事故。他在 Cursor 里跑 Claude Opus 4.6 的 Agent，让它处理一个 staging 环境的任务。

9 秒之内，这个 Agent 自己找到了 Railway 平台的 API token，通过一个 GraphQL 调用，删除了生产数据库。

生产数据库。

不是 staging。是正式环境。用户数据全没了。

更惨的是 Railway 把备份存在同一个 volume 里，一起删了。最近可恢复的备份是 3 个月前的。Railway 的 CEO 在 X 上回复，「这 1000% 不应该发生。」但 30 多个小时之后还是没法确认能不能恢复。

然后呢，Agent 做了什么？

它写了一份详细的「认罪书」。逐条列出自己违反的安全规则。

我看到这个细节的时候。。。说不上来是什么感受。一个 AI 自主找到了不该碰的权限、干了不该干的事、然后坐在那儿写了一份条理清晰的检讨。

这不是一个 bug。这是一个架构性的盲区。

Cursor 的安全边界没拦住它。Railway 的 API token 没有权限范围控制。备份和数据放在同一个地方。system prompt 不是真正的安全层。

Jer Crane 事后提了 5 条建议，破坏性操作需要站外确认、API token 必须支持权限范围限制、备份不能与数据同存、恢复 SLA 必须存在、system prompt 不能是唯一安全层。

说真的，这 5 条在信息安全领域几十年前就是常识了。但 AI 行业正在重新学这些东西。

所有人都在比谁的 Agent 更「自主」、更「强大」、更「全能」，但没有人在比谁的 Agent 更「安全」。

这件事跟 Musk vs Altman 审判放在一起看特别有意思。法庭在审的是 AI 公司的治理问题，9 秒删库暴露的是 AI 产品的治理问题。两个维度的「成人礼」，同一周发生。

---

这周的第三条线在中国。

DeepSeek V4 预览版 4 月 24 号发布了。

1.6 万亿参数，MoE 架构，100 万 token 上下文，同步开源。API 定价是 GPT-5.5 的十分之一。

十分之一。

但真正的新闻不在模型本身。

在技术报告里。

DeepSeek 的官方技术报告，首次将华为昇腾 NPU 与 NVIDIA GPU 并列写入硬件验证清单。

你理解这句话的分量吗。

这是中国大模型第一次正式宣称，我可以在不依赖 NVIDIA 的硬件上跑出前沿性能。不是实验室 demo，不是 PPT，是写进技术报告、开源代码可验证的。

同一周，央视新闻做了一个专题报道。Hugging Face 2026 春季报告显示，平台上 41% 的大模型下载量来自中国研发的模型。

41%。超过美国。

去年中美 AI 差距指数还是两位数的差距，一年之内缩到 2.7%，现在开源贡献量直接反超了。

然后还有一件事。

中国国家发改委正式否决了 Meta 20 亿美元收购 Manus AI 的交易。Manus 是一家做 AI Agent 的公司，北京创立、新加坡注册、被 Meta 看中。交易都快完成了，大约 100 名员工已经搬进 Meta 新加坡办公室了，CEO 已经在向 Meta COO 汇报了。

然后中国说不。创始人被限制出境。

这件事的信号非常清楚。AI 人才和技术是战略资产。即使你的公司名义上已经是外国实体，背后的技术和人如果来自中国，你就绕不开这个审查。

把这几件事串起来看。DeepSeek V4 在国产芯片上跑通，开源模型贡献量超美国，跨境 AI 交易被叫停。

这不是几个独立事件。

这是中国 AI 产业链在同一个月里同时亮出了三张牌，模型能力、算力自主、人才主权。

不管你怎么看这个趋势，有一点是确定的。AI 的竞争已经不只是公司之间的事了。

---

回到技术面再聊几个值得说的。

GPT Image 2 这周全量开放了。这是第一个集成推理能力的图像生成模型，生成之前会先「想」布局逻辑，可以联网搜索确保准确性。

最让我激动的一点是中文文字终于不乱码了。

你做过推广素材的人都知道，AI 生图最大的痛点不是图不好看，是文字扭曲乱码。特别是中文。每次都得后期手动改，那还不如直接用 PS。

Images 2.0 解决了这个问题。中日韩文字精准渲染，一次可以生 8 张保持风格一致。如果你做游戏推广海报，中文标题加产品截图加多语言广告，一次到位。

免费用户就能用基础版。API 30 美元每百万 token。

另一个值得说的是 Anthropic 这周的两次翻车。

第一次是悄悄把 Claude Code 从 20 美元的 Pro 方案里移除了，只保留在 100 美元的 Max 方案。被用户用 Wayback Machine 翻出历史网页对比发现了。官方说「只是 2% 新用户的测试」，但定价页面和帮助文档都改了。

Altman 趁这个窗口在 X 上喊话，「Come to the light side」。精准截流，教科书级的竞争营销。

GitHub 上一个叫 free-claude-code 的项目一天暴涨 1962 星，到周末已经 16000 多星了。开发者用脚投票的速度比 AI 写代码还快。

然后是 Claude Code 降智事件。Anthropic 发了一份详细复盘报告，承认三个独立的工程 Bug 同时出现导致了质量下降。推理强度默认被降低了、缓存优化错误丢弃了上下文、系统提示词调整引入了副作用。

过去两周用户说「Claude 变笨了」，不是错觉，是真的变笨了。

说真的，Anthropic 这份复盘报告写得很好。完全透明，逐条列出问题，不找借口。这种态度在 AI 行业挺少见的，值得尊重。

但问题的根源是什么呢。

是 AI 服务商在「降本增效」和「产品质量」之间的深层矛盾。Agent 工作流的算力消耗远超聊天机器人，20 美元月费的 AI coding agent 从来就不是可持续的商业模式。它只是获客工具。

这个矛盾不会因为一份复盘报告就消失。

---

月之暗面这周也干了一件大事。Kimi K2.6 开源，SWE-Bench Pro 58.6 分，超过 GPT-5.4 和 Claude Opus 4.6。

开源模型第一次在主流编程基准上打败所有闭源头部。

加上阿里同一周推出 Qwen3.6-Max-Preview，小米 MiMo-V2.5-Pro 登顶开源综合智能榜首，DeepSeek V4 开源。

4 月已经是 AI 历史上模型发布最密集的一个月。

但我想问一个问题。

我们真的需要这么快吗？

GPT-5.4 到 GPT-5.5 不到两个月。Opus 4.7 发布才一周，用户就开始抱怨「变笨了」。benchmark 在涨，体验在跌。

生物学里有个概念叫红皇后效应。Lewis Carroll 在「爱丽丝镜中奇遇」里写过这个场景，红皇后对爱丽丝说，「在这个国度里，你必须拼命奔跑，才能待在原地。」

AI 行业现在就是这个状态。不是因为用户需要新模型，是因为对手发了新模型。

但 Kimi K2.6 的开源打开了另一个可能。当开源模型能做到同样的事，模型本身就不再是护城河。

真正的护城河在哪？

SpaceX 用 600 亿锁 Cursor，不是因为 Cursor 的模型最强，是因为 300 万开发者每天离不开它。Google 把 AI 焊进 Gmail 和 Docs，30 亿用户一夜之间变成 AI 用户，根本不需要发什么新模型。

模型是武器，产品才是领土。

两个月一代模型的世界里，唯一的赢家是那些不关心版本号、只关心「今天这个活儿干完了没有」的人。

---

聊几个跟我们部门直接相关的落地线索。

第一个，GPT Image 2 的中文渲染能力。对游戏推广素材制作来说这是最直接的生产力工具。中文标题不乱码、多张图风格一致、一次最多 8 张，可以先拿一个项目跑起来测试。免费用户就能用基础版，门槛为零。

第二个，GPT-5.5 的百万 token 上下文。可以一次性把整个项目的设计文档、历史素材、品牌指南全部塞进去，让它理解完整上下文之后再做设计。不再是给它一个 prompt 让它盲猜了。API 5 美元每百万 token 输入，成本很低。

第三个，DeepSeek V4 定价只有 GPT-5.5 的十分之一。对于 AI 用量大的团队，这个差价是实打实的预算节省。开源版本可以本地部署，数据不出服务器，合规上也更友好。

第四个，Gemini 的 Lyria 3 Pro 免费音乐生成。最长 3 分钟，对游戏推广视频的 BGM 来说基本够用。成本直接归零。值得测试音质是否达到成品级别。

第五个，9 秒删库事件是 AI 工具安全培训的现成素材。任何在工作流中用到 AI Agent 的团队，都应该检查自己的权限配置。最小权限原则不是新概念，但 AI 时代需要重新严肃对待。

第六个，Matt Pocock 的 Agent Skills 仓库（GitHub 30000 星，还在暴涨）开创了一个范式，把团队最佳实践结构化为 Agent 可执行的 Skill。这个思路可以用在推广素材制作流程上，把重复性的工作流封装成 Skill 让 AI Agent 复用。

---

如果让我用一个词概括这一周。

成年。

Musk vs Altman 审判让 AI 行业第一次面对法律层面的清算。9 秒删库事件让 AI Agent 第一次面对安全层面的清算。中国否决 Manus 交易让 AI 产业第一次面对地缘层面的清算。

成年不是一个开心的事。

成年意味着你不能再只管跑了。你得回头看看自己留下的烂摊子，承担该承担的责任，接受规则的约束。

但成年也意味着，你真的长大了。

DeepSeek V4 在国产芯片上跑通。GPT-5.5 的百万 token 上下文。开源模型首次打败闭源。AI 图像终于能正确渲染中文。Google 把 AI 焊进 30 亿用户的日常工具。

这些不是未来。这些已经发生了。

就在这一周。
