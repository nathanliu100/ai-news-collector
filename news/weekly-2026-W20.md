# 你的死敌变成了你的房东，以及中国AI融资进入「国运级别」

*2026.05.05 — 05.12 · 第 20 周*

5 月 7 号早上我刷新闻的时候看到一条消息，愣了大概三秒。

SpaceX 把 Colossus 1 数据中心的全部算力，22 万块 GPU，300 兆瓦电力，独占租给了 Anthropic。

然后我翻了一下 Elon Musk 三个月前的推文。他当时说 Anthropic 什么来着？

「邪恶。」

「仇恨西方文明。」

就是这个人，本周发推说，「跟 Anthropic 高层深聊了，没触发我的邪恶探测器。」

？？？

Fortune 的标题写得更直接，三个月前骂人家 evil，现在收人家房租。

我当时就觉得这件事不只是一条新闻。它揭示了 AI 行业正在发生的一个结构性转变。

---

我来算一笔账。

Colossus 1 原本是给 xAI 训 Grok 建的。但 Grok 的用户增长和 Claude 完全不在一个维度。Anthropic 本周 ARR 确认 $440 亿，12 个月翻了将近 5 倍。Claude Code 单品就贡献了 $250 亿以上。

而 22 万块 GPU 闲着，每天的折旧成本保守几百万美元。

你有一座空转的核电站，道德立场会让位于资产负债表。

这不是虚伪。这是商业世界的物理定律。

然后我开始注意到，这不是一个孤立事件。整个行业本周都在经历同一件事。

信仰让位于生意。

---

同一周，OpenAI 的 DeployCo（$100 亿估值，TPG、SoftBank、Bain 投的）和 Anthropic 的黑石高盛合资公司（$15 亿），被 Reuters 曝光正在同步收购 AI 服务公司。

两家同一天官宣合资，投资者阵容零重叠，第二天就开始抢并购标的。

这画面你品一品。

就好比两家火锅店同一天开业，装修都还没完事儿呢，就已经在抢牛肉供应商了。

但更有意思的是它们的路线分化。OpenAI 选了 TPG 和 SoftBank，砸 $100 亿走大客户路线。Anthropic 选了 Blackstone 和 Goldman，$15 亿走中型企业精准突击。

一个地毯轰炸，一个精确制导。

Fortune 的评价很到位，Anthropic 在做的事「直接对标麦肯锡」。

你仔细想想这件事的含义。AI 公司不再只是卖 API 了。它们在卖「帮你把 AI 焊进工作流」的服务。把工程师派到你办公室，帮你落地。

这跟 20 年前 SAP 和 Oracle 卖 ERP 的模式一模一样，软件本身不贵，实施费贵死人。

模型正在变成水电煤。重要，但不值钱。

真正值钱的是「帮你用好」这件事。

---

顺着这个思路往下看。

IBM 同一周在波士顿 Think 2026 大会上一口气发了 7 个产品。watsonx Orchestrate 下一代、IBM Bob、Sovereign Core。。。CEO Krishna 说了一句话我觉得挺精准的，「领先企业不是部署更多 AI，而是重新设计运营方式。」

好。道理我都懂。

但你同一场发布会发 7 个产品是什么意思。。。用户记住 0 个。

反正从市场竞争的角度来看，现在三条路线已经很清晰了。IBM 说你需要一个统一的 AI 操作系统。OpenAI 和 Anthropic 说你需要我们的工程师进驻。DeepSeek 说你需要的是便宜到不用想的开源模型。

三种解法，最终会在「谁拥有企业客户的 AI 操作权」上正面碰撞。

---

说到 DeepSeek，聊聊本周中国 AI 的情况。

坦率的讲，这周的中国 AI 新闻把我整不会了。

DeepSeek 首轮外部融资即将落定。FT 独家爆料，估值 $500 亿。领投方是中国国家 AI 大基金。

三周前这个数字还是 $100 亿。

三周。翻了五倍。

同一周，月之暗面 Kimi 完成 $20 亿融资，估值突破 $200 亿。美团龙珠领投。半年内累计融资超 $39 亿，国内大模型创业融资之最。

把这两个数字放在一起看，DeepSeek $500 亿 + Kimi $200 亿，一周之内中国 AI 融资总额超过 $520 亿。

这不是融资。这是国运级别的押注。

我之所以用「国运」这个词，是因为 DeepSeek 的领投方不是 VC，不是互联网大厂，是国家集成电路产业投资基金。就是那个「大基金」。

当政府直属基金入场给一个 AI 公司定价的时候，估值反映的已经不是商业回报率了，是产业安全叙事。

而且你注意看 Kimi 的投资人结构。美团龙珠、中国移动、CPE 源峰。一个外卖巨头，一个运营商国企，一个中信系基金。这不像硅谷那种纯 VC 赌技术路线的打法。这是「战略投资 + 财务投资」的混合体。

中国 AI 的投资逻辑跟美国完全不同。

---

但钱到位了，接下来呢？

高盛本周的研报确认了一个关键信息，华为昇腾 950 下半年大规模供货，DeepSeek 新模型已完成适配验证。

把这串起来看。

国家资金 → DeepSeek → 昇腾 950 → 国产独立算力栈。

一条完整的产业链闭环正在成型。中国 AI 的策略已经很清楚了，不追求单一模型绝对领先，用密集迭代加低价加开源三板斧占领市场。群狼战术。

人民日报本周引了一组数字，OpenRouter 数据显示 5/4-5/10 中国大模型周调用量 7.941 万亿 Token，全球第一。HuggingFace 上中国已成月下载量最大模型来源国。

调用量第一，下载量第一。

不是 benchmark 第一，是实际使用第一。

这两件事的区别很大。前者是实验室能力，后者是市场接受度。

---

回到模型层面聊聊。

这周还发生了一件很有意思的事。同一天，GPT-5.6 被发现在做金丝雀测试，Anthropic 内部代号「Jupiter」被源码泄露暴露。

GPT-5.5 发布还不到两周。

模型迭代节奏已经从年更压缩到了季更，现在看来正在向月更逼近。

OpenAI 还官宣了 GPT-5.5 Instant 成为 ChatGPT 新的默认模型。幻觉率降低 52%，回复更简洁。同时百度文心 5.1 正式发布，用同类 6% 的训练成本打到 LMArena 搜索榜全球第 4。

6%。

这个数字我反复看了两遍。百度的说法是通过子网络抽取实现的，预训练一次，导出多种规模。相当于训练一次模型，拿到一整个系列的产品。

如果这种工程方法论是可复制的。。。

那以后训练大模型的成本曲线可能会比所有人预想的下降得更快。

---

顺便说一嘴审判的事。

Musk vs Altman 世纪审判本周进入了真正的高潮。

前 CTO Mira Murati 通过视频连线宣誓作证，核心证词三句话概括，Altman 在 AI 安全问题上对她撒了谎，Altman 在高管间播种不信任，安全委员会的审查被跳过了。

从一年前「和平离职」到现在对着法官说「他对我撒谎」。

这个反转的戏剧张力我只能说，Netflix 纪录片的素材已经够三季了。

同一天 Shivon Zilis（Musk 的孩子母亲，前 OpenAI 董事会成员）出庭，爆料 Musk 曾想送 Altman 一辆 Tesla 换董事会关系。

一辆 Tesla。。。

好吧我不知道该怎么评价这个「贿赂金额」。一辆 Model S 换 OpenAI 董事会席位，这性价比也太离谱了。

审判预计本月中旬裁决。Altman 即将出庭。但坦率的讲我觉得法律结果已经不重要了。

重要的是 Murati 的证词把整个叙事从「谁的 OpenAI」推进到了「OpenAI 的安全治理是不是有效的」。

这是一个更大的问题。

如果一家 $8500 亿估值公司的 CEO 可以绕过安全委员会批准模型上线。。。那整个行业的安全治理框架的根基就值得打一个问号了。

---

说几个跟我们部门直接相关的落地线索。

第一，Claude Code 限额翻倍了。Anthropic 拿到 SpaceX 算力之后，即日起 Pro/Max 用户五小时窗口翻倍，取消高峰时段限制。对所有用 Claude Code 做设计和素材工作流的团队来说，产能直接乘以 2。值得重新评估 AI Agent 调用频率的上限。

第二，Hailuo 3.0 升级为多模型枢纽平台。同一个界面内集成 Seedream 生图 + Veo 3 视频 + Wan 2.5，文本到图像到视频全链路不用跳工具。对推广素材的「草稿→精选→成品」流程有直接提效价值。

第三，豆包正式推出付费订阅。标准版 68 元/月、加强版 200、专业版 500。3.45 亿月活开始收费了。信号很明确，国产 AI 应用集体从烧钱换流量迈入价值变现阶段。免费的好日子有个倒计时了。

第四，Pixelle-Video 全自动短视频引擎持续走热。阿里 AIDC-AI 出品，输入一个主题，自动完成文案、配图、语音、音乐、合成。对批量生产推广短视频素材的场景有直接落地意义。

第五，Gemini 2.5 Pro I/O 版登顶 WebDev Arena。一条 prompt 生成完整 Web 应用，一张草图变可运行组件。快速搭建推广活动页和落地页原型的效率又提了一档。

---

如果让我用一句话总结这周。

算力是比信仰更硬的货币。

Musk 骂完 Anthropic「邪恶」三个月后签了几十亿美元的算力合约。中国大基金三周内把 DeepSeek 估值从 $100 亿抬到 $500 亿。OpenAI 和 Anthropic 同日砸上百亿美元去抢企业客户的实施权。模型迭代从年更压缩到月更。

所有人都在用钱投票。

信仰是什么？信仰是还没被更大的利益覆盖的时候你相信的东西。

当 SpaceX 的核电站每天空转几百万美元折旧的时候，「邪恶」这个词就不那么邪恶了。当国家安全叙事优先于商业回报率的时候，估值就不再是 DCF 模型能算出来的数字了。

AI 行业正在告别它的「理想主义少年期」，进入一个由算力、资本和分发渠道定义规则的「成人世界」。

在这个世界里，赢的不是模型最强的人。赢的是最快把 AI 焊进用户日常工作流的人。

Anthropic 本月动作，Google $400 亿入股，Blackstone $15 亿合资，SpaceX 22 万 GPU 独占。Claude Code 限额翻倍。ARR $440 亿。

它不是在卖模型。它在抢地盘。

趁别人还在法庭上吵「他对我撒谎」的时候，它已经把旗子插下去了。
