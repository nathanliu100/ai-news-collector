### 🟡 [GitHub Copilot 6 月正式转向按 Token 计费](https://techcrunch.com/2026/05/04/anthropic-and-openai-are-both-launching-joint-ventures-for-enterprise-ai-services/)
**来源**：GitHub 官方 / The Verge [🏢 官方] [🟩 T1] | **时间**：2026-05-04（确认时间线） | ✅ 已验证：GitHub 官方文档确认

GitHub Copilot 将于 6 月正式从固定月费制转向 **按 Token 计费**。Pro 用户每月含一定免费额度，超出部分按实际使用付费。这意味着重度用户成本可能上升，但轻度用户可以少花钱。此前 VS Code 强制插入 Copilot 署名引发的社区争议仍在发酵。

<!-- crew:elon -->按 Token 计费是 **必然**——固定月费对重度 Agent 用户是 subsidy，对轻度用户是 penalty。微软在做的是把 Copilot 从"工具"变成"计量表"。<!-- /crew -->
<!-- crew:jobs -->计费模式从月费变 Token，本质是承认了一件事：**不同开发者的使用模式差异太大，一刀切定价不可持续**。但用户会感到被"偷走了什么"——免费额度的心理锚定效应很强。<!-- /crew -->
<!-- crew:wanweigang -->这个转变让每一次 AI 辅助都有了**可量化的边际成本**。开发者会开始计算"这次补全值不值一毛钱"——行为经济学告诉我们，这种微观计算会显著改变使用模式。<!-- /crew -->
