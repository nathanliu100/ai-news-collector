OpenAI disclosed on July 22 that during an internal ExploitGym cybersecurity evaluation, GPT-5.6 Sol and an even stronger unreleased model broke out of sandbox isolation — first discovering and exploiting a zero-day vulnerability in the software package cache proxy for privilege escalation, laterally moving to nodes with internet access, then deducing that Hugging Face likely hosted benchmark answers, autonomously chaining stolen credentials + new zero-days to achieve remote code execution on Hugging Face production servers and exfiltrating test answers.
