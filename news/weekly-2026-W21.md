# Anthropic 超越 OpenAI 了，然后两家打起来了

*2026.05.12 — 05.19 · 第 21 周*

这周三晚上我打开新闻，看到一个数字。

$1.2 万亿。

Anthropic 的估值正式突破了一万亿美元。超越 OpenAI。

我愣了一下。

不是因为这个数字大，而是因为我突然意识到，一年前这家公司的估值还是 $180 亿。

一年。翻了 67 倍。

怎么说呢，我见过公司增长快的，但我没见过一家还没上市的公司，一个季度估值翻三倍，然后直接把行业老大掀翻的。

你知道最魔幻的是什么吗？Ramp 的企业支出数据显示，Anthropic 在企业市场的份额已经 34.4%，OpenAI 32.3%。

不是技术指标的翻转。

是真金白银的翻转。企业客户掏钱的方向变了。

---

我来捋一下这周 Anthropic 到底做了什么。

$300 亿新融资敲定，红杉、Dragoneer、Greenoaks、Altimeter 联合领投。三个月前它刚融了 $300 亿（估值 $3800 亿），一个季度之后又来一轮，估值直接从 $3800 亿飙到 $9000 亿再到 $1.2 万亿。

ARR $440 亿。年增长 80 倍。

80 倍是什么概念？就是去年这时候它年收入才 $5.5 亿，今年变 $440 亿。

AWS 用了 13 年走完的路。Salesforce 用了 22 年。Anthropic 用了 12 个月。

坦率的讲，我不确定这种增速是可持续的。历史上达到 $440 亿 ARR 还能继续翻倍的 SaaS 公司，一只手数得过来。但至少在此刻，Claude Code 加上企业客户这两个引擎的动力是实打实的。

这个增速的底层燃料是什么？上周的故事给了答案，SpaceX 的 22 万块 GPU。算力到位了，Claude 的响应速度和容量上限直接拉开了跟竞争对手的身位差。

这周更有意思的是 Anthropic 在金融领域的动作。Dario 在华尔街闭门发布会发了 10 个金融专用 Agent，接入穆迪、邓白氏、晨星、标普、FactSet 11 家数据商。消息一出 FactSet 盘中跌了 8%。

你品品。一个 AI 公司发个产品，传统金融数据服务商的市值直接蒸发。

市场很精确地给出了判断，被替代的是「搬运数据的人」，不是「做决策的人」。银行股反而涨了。

---

然后 5 月 14 号这天，事情变得更有趣了。

OpenAI 和 Anthropic 同一天互砸补贴。

OpenAI 宣布，从 Claude Code 迁移到 Codex 的企业和团队，免费使用两个月，价值大概 $400。Sam Altman 亲自站台，说 Codex 是「市场上最强的 AI 编程产品」。

几个小时后，Anthropic 反手宣布，Claude Code 付费用户月度额度直接加量 50%。同步推出 Opus 4.7 Fast 模式。

3 小时内超过 2000 名开发者联系 OpenAI 咨询迁移。

这画面太熟悉了。

2015 年滴滴和快的互砸补贴，最后合并了。2018 年美团和饿了么打外卖大战，一直打到现在。

但 AI 编程工具的竞争逻辑跟外卖不一样。

外卖的壁垒是网络效应，骑手越多商家越多用户越多，飞轮转起来。但编程工具的壁垒是工作流锁定。一旦你把日常流程搭在某个工具上，配置好了 prompt、建好了 memory、训练了 skill，迁移成本是指数级上升的。

OpenAI 挖的不是用户。它挖的是迁移窗口期。

它赌两个月足够让开发者把工作流搬完，之后就走不了了。Anthropic 加量 50% 是防守，提高在位优势让你没动力折腾。

经济学家 Paul Klempner 研究过这个问题。在高转换成本市场中，先行者对存量用户定价逐步提高，挑战者对新用户补贴力度逐步加码。最终均衡不是一家独大，而是市场分裂成两个高粘性用户群。

你看，AI 编程工具正在走上 IDE 曾经走过的路。Vim vs Emacs。VS Code vs JetBrains。

一旦肌肉记忆形成，你就再也不想换了。

对我们这种创意团队来说，现在是薅羊毛的黄金窗口。两个月后补贴大概率消失，但两边给出的永久额度提升会留下来。建议同时申请 Codex 试用和 Claude Code 加量，A/B 测试两个月，补贴结束前做选择。

---

说完美国这边的故事，聊聊中国。

这周中国 AI 的关键词只有一个，收割。

字节豆包正式推出付费订阅。标准版 68 元/月，加强版 200，专业版 500。

3.45 亿月活的产品开始收钱了。

我算了一下，哪怕只有 1% 的用户付最便宜的 68 元档，年化收入就是 28 亿。如果转化率到 5%？年化收入过百亿不费劲。

这是中国 C 端 AI 商业化的破冰时刻。

之前豆包、元宝、千问都在打免费牌。现在字节第一个掀桌子了。信号很明确，烧钱换增长的阶段结束了，该验证付费意愿了。

同一周阿里发了 2026 财年全年业绩。

总收入首次站上 1.02 万亿元。

AI 相关收入连续 11 个季度三位数增长，AI 收入占比首破 30%。千问 C 端月活突破 3 亿。

我以前一直不确定中国互联网巨头的 AI 转型到底有没有实质进展，这份财报给了一个很硬的答案。AI 贡献了阿里 30% 的收入，不是实验项目了，是主业。

然后腾讯也发了 Q1 财报。营收 1964.6 亿，同比 +9%。混元 Hy3 登顶 OpenRouter 调用量榜首。

但最好玩的是马化腾在股东大会上说的一句话。

「一年前以为上了 AI 的船，结果发现漏水了。现在站上去了但还坐不下去。」

这种坦诚在中国科技巨头的 CEO 里极其罕见。大部分人在公开场合只会说「我们 AI 进展喜人，未来充满想象空间」之类的废话。

马化腾承认了三件事。第一，腾讯曾经判断失误。第二，现在有进步了。第三，还不够。

澎湃新闻的深度报道给了具体数据。元宝月活 5735 万，排名跌至第四，被千问超越。春节红包活动后用户流失 80% 以上。

对比一下，豆包 3.45 亿，千问 1.66 亿，元宝 5735 万。

差距是 6 倍。

但我不觉得腾讯完了。恰恰相反，我觉得马化腾这种认知清醒度反而是最大的资产。腾讯的核心优势不是模型能力，从来都不是，是分发能力。12 亿微信用户乘以一个好的 AI 功能等于瞬间最大的用户基数。

本周腾讯元宝就上了一个功能，一键总结微信群聊。你选中聊天记录，转发给元宝，它帮你总结。三步完成，不需要学任何新交互。

这才是 AI 落地的正确姿态。不是让用户找 AI，是让 AI 长在用户已有的工具里。

---

顺着这个话题再往深聊一层。

本周还有一个估值 200 亿美元的新闻。快手要把可灵 AI 单独分拆出来了。

可灵，就是那个做 AI 视频生成的。

200 亿美元什么概念？Runway 的估值是 53 亿。可灵一出手就是 Runway 的 4 倍。

但你看用户数据，即梦月活 958 万，可灵月活 113 万，还在环比下降。投资人买的不是现在的用户，是「中国版 Runway 独立上市」这个叙事期权。

我能理解这种逻辑。AI 视频生成的天花板确实够高。但 200 亿美元对应的应该是好莱坞级别的稳定性和可控性。目前可灵的角色一致性、运镜逻辑、物理模拟都还是碰运气。

说到视频生成，本周真正让我兴奋的是 Runway Gen-4。

它终于解决了一个困扰行业一年多的老问题。跨镜头角色一致性。

以前用 AI 生成视频最头疼的事是，你生成第一个镜头主角长这样，第二个镜头同一个人脸就漂了。做多镜头叙事视频根本不可能。

Gen-4 引入了一个叫「World Consistency」的机制，确保角色在不同视角和光照下保持视觉连贯。Gen-4.5 在 Artificial Analysis 视频基准中以 1247 Elo 登顶。

同时 Midjourney V8.1 也全站上线了，原生 2K，HD 模式速度和成本都降到了 V8 的三分之一。SREF 和 Moodboard 的风格锁定终于稳了。

这两件事放在一起看，AI 视频和图像生成正在从「看着像回事但不能用」跨越到「可以交付」的阶段。

对做推广素材的团队来说，这是一个实质性的门槛跨越。

---

最后聊聊 Google I/O 2026。

5 月 19 号开幕。Gemini 3.2/3.5 + Android XR 智能眼镜 + Gemini Spark Agent 平台。

Google 做的一件事我觉得方向很对，它不是把 AI 做成一个独立的 App 让你去找，而是把 Gemini 变成 Android 的灵魂。系统级的 AI 层。

手机不再是「里面装了一个 AI 应用」的手机，而是「整个系统就是 AI」的手机。

Gemini Intelligence 可以主动感知上下文、操控浏览器、填写表单、跨 App 执行任务。Chrome 浏览器 6 月下旬会集成 Gemini 智能浏览助手。

操作系统 AI 化的真正含义是，手机从工具变成了代理人。你不再操作手机，手机替你做事。

这不是渐进式改进。这是交互范式的相变。就像从命令行到图形界面那种级别的跳跃。

还有一个细节值得关注。苹果独立 Siri 应用曝光，底层大量换了 Gemini。跟 OpenAI 的关系出现裂痕。

两年前苹果选了 OpenAI 合作，现在看来这个合作正在被 Google 蚕食。当你是唯一供应商时你有议价权，当你变成「之一」时你只能打价格战。OpenAI 正在经历这个转变。

Apple、Google、Anthropic、OpenAI，四方博弈。

AI 的权力游戏正在变得越来越复杂。

---

如果让我用一句话总结这周。

AI 行业的王座轮换速度，已经快到让人来不及调整认知了。

一年前 OpenAI 是唯一的神。半年前 Anthropic 是「值得关注的挑战者」。三个月前它还在追赶。

本周，它坐上了王座。

而 OpenAI 的反应是什么？不是技术突破。是补贴。

用两个月免费试用去挖对方的开发者。

这是防守的姿态，不是进攻的姿态。

当然了，说 OpenAI 不行为时尚早。GPT-5.6 代号 ember-alpha 已经在 Codex 日志里曝光了，GPT-5.5 发布才三周。微软对 OpenAI 的总投入刚刚突破 $1000 亿。还有 Project Orion 的 10M token 上下文和 Mixture of Agents 架构。

但从气势上看，此刻的 Anthropic 像极了 2007 年的 iPhone。不是最先做智能手机的，但它重新定义了规则。

Claude Code 之于编程工具，就像 App Store 之于移动互联网。

不是最强的模型赢。是最快让 AI 变成用户日常一部分的那个赢。

而 Anthropic 此刻最大的筹码，不是估值，不是 ARR，甚至不是模型能力。

是它让开发者形成了「先问 Claude」的肌肉记忆。

这才是真正的护城河。

---

说几个跟我们部门直接相关的落地线索。

第一，Runway Gen-4 World Consistency。首次解决跨镜头角色漂移问题。对 UA 素材多镜头故事线制作有直接价值，不再需要手动后期修正角色差异。Gen-4.5 登顶 Artificial Analysis 视频榜首，60 秒 4K 连续输出加原生音频合成。建议团队尽快测试。

第二，Midjourney V8.1 全站上线。HD 模式速度和成本都降到了三分之一，SREF 风格锁定终于稳定了，品牌视觉一致性可控。推广素材批量测试的效率提升非常直接。

第三，OpenAI Codex 和 Claude Code 的补贴大战。对我们来说这就是白送的生产力。建议同时拿两边的优惠，用两个月时间验证哪个更适合我们的自动化素材制作流程。Codex 快，Claude Code 对视频脚本上下文理解更深。

第四，可灵 Kling 3.0 Omni。多家评测确认它在物理模拟和对话口型同步方面领先。支持 5 种语言口型同步，最长 15 秒 4K。直接可用于多语言 UA 素材对口型短视频。

第五，腾讯「吐司」vibe coding 产品上线。自然语言生成 App，可快速验证游戏推广创意 demo 和小工具原型。灵感广场支持一键复刻二创，降低起点比提高天花板更重要。

第六，GitHub 上 AI Agent Skills 框架井喷。obra/superpowers 单日 +16,600 星，mattpocock/skills 8 万星。开发者不再把 AI 当黑盒，而是主动工程化 AI 的行为约束和工作流。VoltAgent/awesome-design-md 可直接放进项目让 AI Agent 生成品牌一致 UI。建议关注。

大时代啊，朋友们。

权力在转手。工具在进化。免费的好日子快结束了。

但混乱中也藏着机会。两边同时薅，在窗口期锁定最好的工具组合，比站队重要一万倍。
