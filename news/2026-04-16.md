### 🔴 [Claude Code Routines 上线：定时任务 + API + GitHub 触发](https://code.claude.com/docs/en/routines)
**来源**：Anthropic 官方 [🏢 官方] [🟩 T1] + 36 氪报道 [📰 媒体] [🟩 T1] | **时间**：2026-04-15 | ✅ 已验证：Claude Code 官方文档 + 36 氪
Anthropic 为 Claude Code 推出 **Routines** 功能（Research Preview），将 Claude Code web 版的云端 session 打包为可自动执行的任务：支持 **Cron 定时触发**、**API 调用触发**、**GitHub 事件触发**三种模式。用户只需定义 prompt、repo 和 connector，Claude 即可在 Anthropic 托管的基础设施上自主运行。Pro/Max/Team 套餐均可使用。这让 Claude Code 从编码助手进化为「云端自动员工」。[💎 部门相关]
